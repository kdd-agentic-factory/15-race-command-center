# Paper Dashboard Description

The dashboard is the paper-facing evidence surface. It presents the complete
technical stack and the operational utility of the system:

- Live telemetry and latency indicators.
- EKF state-estimation quality.
- Corner micro-segmentation.
- Tire degradation predictions.
- Crew-chief recommendations and approval trail.
- Circuit-specific parts evidence.
- Pattern discovery explanations.
- Infrastructure observability from Prometheus, Grafana and OpenTelemetry.
