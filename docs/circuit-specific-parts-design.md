# Circuit-Specific Parts Design

The parts design studio demonstrates that the architecture does more than
analyze data: it closes the engineering loop.

Supported part families:

- Brake cooling ducts.
- Winglet profiles.
- Side flow deflectors.
- Aero covers.
- Lightweight brackets.
- Gyroid lattice structures.
- High-downforce circuit bodywork.
- Low-drag circuit bodywork.

The evidence chain is:

```text
telemetry -> pattern -> hypothesis -> part design -> validation -> report
```
