# Pre-Grand Prix Process

Inputs:

- Circuit profile.
- Historical telemetry.
- Expected temperature.
- Asphalt abrasiveness.
- Critical corners.
- Braking zones.
- Acceleration zones.
- Available parts.
- Regulatory constraints.
- Base setup.
- Tire strategy.

Process:

```text
Pre-GP Input
  -> Circuit Profile
  -> Historical Pattern Retrieval
  -> Setup Baseline
  -> Tire Risk Estimation
  -> Circuit-Specific Parts Recommendation
  -> Crew Chief Review
  -> FP1 Plan
```
