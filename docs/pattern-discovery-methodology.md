# Pattern Discovery Methodology

The pattern discovery tool supports questions such as:

- Which corners show degradation first?
- Which setup reduces spin at corner exit?
- Which suspension changes worsen rear tire wear?
- Which pattern anticipates performance drop-off?
- Which aero parts worked best in stop-and-go circuits?
- Which engine map, tire and temperature combination creates risk?

Techniques:

- Corner-level clustering.
- Anomaly detection.
- Sequential pattern mining.
- Setup-performance correlation.
- Session embeddings.
- RAG/CAG retrieval of similar sessions.
- Agent-assisted causal explanation.
